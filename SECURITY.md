# Security Policy

## Supported Versions

Kami sangat menyarankan Anda untuk selalu menggunakan versi terbaru aplikasi untuk mendapatkan perbaikan keamanan terkini.

| Version | Supported          |
| ------- | ------------------ |
| v1.2.0  | :white_check_mark: |
| < v1.2.0| :x:                |

## Reporting a Vulnerability

Jika Anda menemukan celah keamanan dalam aplikasi ini, harap jangan membukanya sebagai *Public Issue*. Sebagai gantinya, silakan laporkan melalui:

1.  **Email**: [security@neverlandstudio.id](mailto:security@neverlandstudio.id)
2.  **Private Issue**: Gunakan fitur *Private Vulnerability Reporting* di GitHub jika tersedia.

Kami akan menanggapi laporan Anda dalam waktu maksimal 48 jam dan bekerja sama dengan Anda untuk menyelesaikan masalah tersebut sebelum dipublikasikan.

---

## Data Privacy

Aplikasi Smart Shopping dirancang dengan prinsip **Privacy by Design**:
- Semua data disimpan secara lokal menggunakan [Hive](https://pub.dev/packages/hive).
- Tidak ada data yang dikirim ke server eksternal tanpa persetujuan eksplisit dari pengguna.
- Kami menyarankan pengguna untuk menggunakan fitur kunci aplikasi atau biometrik pada perangkat masing-masing untuk perlindungan tambahan.
