# Menjadi Kontributor

Terima kasih telah tertarik untuk berkontribusi di **Smart Shopping**! Kami sangat menghargai bantuan Anda untuk membuat aplikasi ini menjadi lebih baik.

## Cara Berkontribusi

### 🐛 Melaporkan Bug
- Pastikan bug tersebut belum dilaporkan di bagian [Issues](https://github.com/NeverlandStudio/smart-shopping/issues).
- Berikan detail langkah-langkah untuk mereproduksi bug tersebut.
- Sertakan informasi perangkat dan versi Flutter yang Anda gunakan.

### 💡 Mengusulkan Fitur Baru
- Buka *Issue* baru dengan label `enhancement`.
- Jelaskan mengapa fitur tersebut berguna dan bagaimana cara kerjanya.

### ⌨️ Berkontribusi Kode (Pull Request)
1. Fork repository ini.
2. Buat branch baru (`git checkout -b fitur/nama-fitur`).
3. Lakukan perubahan dan commit (`git commit -m 'Menambahkan fitur X'`).
4. Push ke branch Anda (`git push origin fitur/nama-fitur`).
5. Buka *Pull Request* ke branch `main`.

## Panduan Kode
- Gunakan standar linter Flutter (`flutter_lints`).
- Pastikan kode Anda rapi dan memiliki komentar yang jelas.
- Jalankan tes sebelum mengirimkan PR.
